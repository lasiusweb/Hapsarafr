# Technology Stack

This document outlines the core technologies and frameworks utilized in the Hapsara Farmer Registration project.

## Frontend

*   **Framework:** React
*   **Language:** TypeScript
*   **Build Tool:** Vite
*   **Styling:** Tailwind CSS
*   **Routing:** React Router DOM (`react-router-dom`)

## Data Management

*   **Offline-first Database:** WatermelonDB (`@nozbe/watermelondb`) - for robust local data storage and offline capabilities.
*   **Backend Services:** Supabase (`@supabase/supabase-js`) - for authentication, cloud database, and other backend functionalities.

## AI Integration

*   **AI API:** Google Gemini API (`@google/genai`) - for integrating artificial intelligence capabilities into the application.

## Other Key Libraries/Tools

*   `date-fns` - for date manipulation.
*   `jspdf`, `html2canvas` - for PDF generation and image manipulation.
*   `leaflet` - for mapping functionalities.
*   `xlsx` - for spreadsheet processing.
*   `clsx`, `tailwind-merge` - for utility class management.
*   `dompurify` - for sanitizing HTML.
